# event-json-rpc

基于 libevent,以二进制头+json为协议来实现的 rpc 框架.

# 参考和依赖

```
libevent
cJSON
zlog
libzdb
```

## 以此为基础
https://github.com/jasonish/libevent-examples

## json c 解析器
https://github.com/kbranigan/cJSON

## c loger
https://github.com/HardySimpson/zlog

## 数据库连接池
https://github.com/mverbert/libzdb
